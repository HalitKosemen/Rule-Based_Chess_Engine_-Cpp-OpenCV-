## Rule-Based Chess Engine (C++ / OpenCV)

This project is a **rule-based chess engine implemented in C++**, featuring an  
**interactive graphical interface built with OpenCV**.

All chess rules are **explicitly implemented from scratch**, without using any
external chess engines or AI libraries.

The application allows users to **interact with the board using mouse input**,
visualize legal moves, and play a complete game of chess while the engine enforces
all standard rules.

- Board state and attack squares are evaluated in real time
